Lista de Exercícios - JavaScript para Web
Este repositório contém uma série de exercícios práticos desenvolvidos para a disciplina de Programação para Internet (PPI) do Instituto Federal de Santa Catarina.

Sobre o Projeto
Foram desenvolvidos 12 exercícios que abordam conceitos fundamentais de JavaScript para desenvolvimento web. Cada exercício foca em um tópico específico, permitindo o aprendizado gradual e prático da linguagem.

Conteúdos Abordados
1 - Introdução ao JavaScript - Conceitos básicos e execução no navegador

2 - Tipos de Variáveis - Demonstração dos tipos de dados e operadores

3 - Cálculo de IMC - Aplicação com formulários e estruturas condicionais

4 - Gerador de Números - Trabalhando com números aleatórios e arrays

5 - Números Primos - Implementação de algoritmos matemáticos

6 - Verificador de Palíndromos - Manipulação de strings e lógica

7 - Mudança de Cores - Interatividade com eventos de clique

8 - Cores com Teclado - Eventos de teclado e alteração dinâmica

9 - Calculadora - Interface completa com operações matemáticas

10 - Eventos JavaScript - Diferentes tipos de eventos do usuário

11 - Local Storage - Armazenamento local no navegador

12 - Canvas - Desenho e manipulação gráfica

Como Utilizar

Cada exercício está em um arquivo HTML separado numerado de 1 a 12. Para visualizar qualquer exercício:

Clone este repositório em seu computador

Abra o arquivo index.html em seu navegador

Navegue pelos exercícios usando os links fornecidos

Não é necessário instalar nenhuma dependência adicional. Todos os exercícios funcionam diretamente no navegador.
