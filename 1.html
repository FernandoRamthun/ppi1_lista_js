O que é JavaScript?

Motivação: O JavaScript foi criado para tornar as páginas web interativas. Antes dele, as páginas eram estáticas e não respondiam às ações dos usuários.

Execução no navegador vs servidor: No navegador o JavaScript roda no computador do usuário, enquanto no servidor (com Node.js) processa dados antes de enviar ao navegador.

Sandbox no navegador: É um ambiente seguro que impede que o código acesse recursos do sistema operacional, protegendo o usuário.

Objetos globais: document - manipula a página HTML; console - exibe mensagens no console; alert - mostra caixas de diálogo.
